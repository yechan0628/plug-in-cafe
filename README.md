# 🔌 Plug-in Cafe (플러그인 카페)

> **신촌역 인근 실시간 콘센트 및 좌석 현황 관리 웹 애플리케이션**

각 카페의 실시간 콘센트 보유 현황, 좌석 배치도, 혼잡도 및 위치 정보를 한눈에 파악할 수 있는 반응형 웹 플랫폼입니다.

---

## ✨ 핵심 기능
1. **실시간 좌석 배치도 (Live Floor Plan)**
   - 각 카페의 실제 1인석, 2인석(소파), 3인석(스터디 테이블) 면적 비율 고증 및 맞춤형 아이콘 매핑.
   - 2층 규모 복층 카페의 경우, 1층 도면 바로 밑에 2층 도면이 수직 스택으로 동시에 렌더링되어 시각적 편리함 증대.
2. **콘센트 수량 및 상세 현황 시각화**
   - 다인용 좌석에서 제공되는 개별 콘센트의 개수(🔌 x2, x3)를 건축용 뱃지로 시각화.
   - 개별 좌석 클릭 시 사용 중/빈 자리 상태가 실시간으로 반전되며, 좌측 대시보드(🔌 콘센트 좌석 수 및 🪑 전체 좌석 수)에 즉시 동기화.
3. **인터랙티브 SVG 지도 및 마커**
   - 각 매장의 공식 고해상도 로고(이디야, 스타벅스, 할리스, 투썸, 커피빈, 탐앤탐스 등)를 하얀색 핀 마커 원형에 내장하여 가시성 극대화.
   - 마우스 휠 줌(확대/축소) 및 드래그(이동) 가능.
4. **커스텀 필터링 및 검색**
   - 검색창 입력 필터 및 `🔌 콘센트 많음`, `🍃 한적함`, `🚗 주차가능` 태그 원터치 필터 기능.
